On the App Store

A literary, hard–science-fiction civilization roguelite. Guide a species from its first orbital stations toward the energy of an entire galaxy — and when the run ends, in triumph or extinction, the game writes its Chronicle: a short obituary of the civilization you led. Fully offline. No ads, no accounts, no data collection.

Common Questions

My run disappeared.

One run is active at a time. Finishing a run (victory or defeat) retires its save and writes the Chronicle. Force-quitting mid-run is safe — the game saves after every beat and offers Continue on the home screen.

It feels too hard / I keep collapsing.

Each run secretly favors one threat — machines, biosphere, war, or cohesion. Watch which meters climb fastest and spend your decisions easing that pressure. Declaring victory at Kardashev Type II is legitimate; the climb to Type III is meant to be a gamble.

Does it need internet?

No. The game is fully offline and collects no data of any kind.
